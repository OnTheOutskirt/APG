# Ascent Performance Group — Lead Magnet Concepts

Interactive mockups of three lead-magnet concepts for Ascent Performance Group, built as a single self-contained `index.html` (HTML/CSS/JS, no build step).

Open `index.html` in any browser and use the switcher at the top to flip between the three concepts.

## Concepts

1. **Incentive Program ROI Assessment** — A 3-minute, multi-step assessment that generates a benchmarked scorecard (overall score, four pillar ratings, and ROI upside). Includes an internal **admin panel** that defines the scoring rubric: how each answer choice contributes points to each pillar, the pillar weights, the rating thresholds, and the upside ranges.

2. **Executive Benchmark Report** — A gated "2026 Incentive Travel & Recognition Benchmark Report" landing page with a lead-capture form, preview stats (locked/unlocked), and a table of contents.

3. **Incentive Trip Cost Calculator** — An interactive calculator (winners, guests, nights, destination, luxury level) that reveals an estimated investment range only after the visitor submits name, company, work email, and revenue. Includes an internal **admin panel** to tune all pricing factors.

## Notes

- These are front-end mockups. Forms display confirmation messages but do not submit anywhere.
- All benchmark numbers and pricing factors are realistic placeholders, adjustable via the admin panels.
- `apg_logo.png` is the Ascent Performance Group logo used across the concepts.
